# TeevoLink v2 — 纯中文操作手册（给不懂英文的同学）

你不需要看懂所有英文。按本页顺序做，**每一步只做一个框**。  
遇到英文看不懂：**复制整段发给我（或 Cursor 里问我）**，我会用中文告诉你要点哪里。

---

## 一、哪些能改成中文？

| 软件/网站 | 能否中文 | 怎么改 |
|-----------|----------|--------|
| **Chrome / Edge 浏览器** | 能 | 设置 → 语言 → 中文(简体) 放最上 → 重启浏览器 |
| **GitHub 网站** | 部分能 | 浏览器已是中文后，GitHub 左下角或设置里找 Language |
| **Cursor（写代码软件）** | 能 | 见下文「Cursor 改中文」 |
| **GitHub Desktop** | 部分能 | File → Options → 若有 Language 选中文 |
| **Git Bash 黑窗口** | 几乎不能 | 只记 3 条命令即可（下面有） |
| **Vercel 网站** | 英文为主 | 用本手册「看到什么点什么」（中英对照） |

---

## 二、Cursor 改成中文界面

1. 按键盘 **`Ctrl + Shift + P`**（打开命令面板）
2. 输入：`Configure Display Language`
3. 选 **中文(简体)** / `zh-cn`
4. 重启 Cursor

之后菜单大多是中文，效率会高很多。

## 三、你只需要记住的 Git 命令（复制即可）

在项目文件夹 `teevolink-v2` 里，以后只可能用到：

```bash
cd "D:/XILU文件/Cyrus Cursor/teevolink-v2"
git add .
git commit -m "更新"
git push
```

含义（不用背英文）：
- 保存改动到本机
- 上传到 GitHub

**推送成功** = 终端里没有红色 `fatal`、`error`。

## 五、Vercel 部署（中英对照 — 照着点）

目标：得到一个 **https://xxxx.vercel.app** 网址，用 Chrome 测鼠标。

### 5.1 登录

1. 打开：https://vercel.com  
2. 点 **Sign Up** 或 **Log In**（注册/登录）  
3. 选 **Continue with GitHub**（用 GitHub 登录）  
4. 用账号 **xilu7** 授权

### 5.2 导入项目

1. 登录后点 **Add New…** → **Project**（添加新项目）  
2. 列表里找到 **teevolink**，点 **Import**（导入）  
3. **Framework Preset** 应显示 **Vite**，不用改  
4. **Build Command** 填：`npm run build`（一般自动有）  
5. **Output Directory** 填：`dist`（一般自动有）  
6. 点 **Deploy**（部署）  
7. 等 1～3 分钟，出现 **Congratulations** 或绿色勾 = 成功

### 5.3 打开网站

1. 点 **Visit** 或 **Domains** 下面那一行 `https://teevolink-xxx.vercel.app`  
2. 用 **Chrome**，插上鼠标，点页面上的连接按钮  
3. 浏览器问「是否允许访问设备」→ 点 **允许**

### 5.4 常见英文 → 中文意思

| 英文 | 意思 |
|------|------|
| Deploy | 发布/部署 |
| Building | 正在构建 |
| Ready | 好了，能访问 |
| Failed | 失败，把日志截图发我 |
| Environment Variables | 环境变量（v2 一般不用填） |
| Redeploy | 重新部署 |

## 六、想用 teevolink.teevolution.gg 这个旧网址？

域名绑在**同事的 Vercel** 上时，你要：

- 让同事把域名改绑到你的 Vercel；**或**
- 先用 `xxx.vercel.app` 测试，域名以后再说

需要发给同事的话（复制）：

> 请把 teevolink.teevolution.gg 域名绑到我的 Vercel 账号（xilu7），或把原 Vercel 项目转给我。
