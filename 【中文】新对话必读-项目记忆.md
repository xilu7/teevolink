## 三、硬件与连接（必须遵守）

- **传感器**：PAW3950；**主控**：nRF52840  
- **DPI**：50–42000，步进 50；推荐档 400/800/1600/3200/6400  
- **LOD**：低 0.7 / 中 1 / 高 2 mm  
- **按键**：6 键（含 DPI）  
- **三模**：蓝牙 / 2.4G（RapidSync **VID `0x3554`**）/ USB  
- **网页调参仅支持**：**2.4G + RapidSync 接收器** 或 **USB 有线**（蓝牙网页一般不可用）

**用户操作要点**：Chrome/Edge → 首页点连接 → 弹窗选 RapidSync → 允许 → 鼠标切 2.4G → 晃动唤醒 → 设备页点「同步设备」直到 **已连接**。

**页脚版本号**（用于确认是否最新部署）：在 `HomeView.vue` 的 `BUILD_TAG`，当前应为 **`2026-06-04-f`**。首页含设备参数卡 + 四步连接引导；再次进入自动 `Get_HistoryDevicesInfo` 回连。

**顶栏「20%」**：一般是**鼠标电量**，不是 Cursor 百分比。
