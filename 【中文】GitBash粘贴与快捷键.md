# Git Bash：用 Ctrl+V 粘贴

## 方法一（推荐）

1. 在 Git Bash 窗口 **标题栏空白处** 点 **右键**
2. 选 **Options（选项）**
3. 左侧点 **Keys（按键）**
4. 勾选：**Ctrl+Shift+Letters shortcuts** 或 **Ctrl+V paste**（不同版本文字略有不同，有「paste」就勾）
5. 点 **Save（保存）**，关掉窗口重新打开 Git Bash

## 方法二（不设置也能用）

- **鼠标右键** = 粘贴  
- **Shift + Insert** = 粘贴  

## 在 Cursor 里的终端

- 一般用 **Ctrl+V** 即可  
- 若提示「是否粘贴多行」，点 **粘贴** 或 **Paste**
